## Como rodar este projeto

### Pré requisitos

- ter o java 8 instalado

### Instalação

- Clone este projeto
- Entre na pasta raiz e construa o projeto

```
Linux
./gradlew clean install

Windows
gradlew.bat clean install
```
 
### Rodando os testes
```
Linux
./gradlew test

Windows
gradlew.bat test
```

 
### Rodando o projeto localmente

```
Linux
./gradlew bootRun

Windows
gradlew.bat bootRun
```
